# Organic chain
It is a web3 application connected to IoT devices on the farms. It is developed to make agriculture and lifes associated with it better. 

It allows farmers to tokenize there crops even before sowing them, which enables them to have captial for agricultural expenditure at the start of sowing season. Organic Chain also opens an attractive investment oppurtunity as agriculture has given more profit that any other industry in last decade. And at last it also solves the problem to find a suitable market for organic produce.

### 2. Import few of accounts from Ganache to Metamask for using them during testing.

```shell 
    truffle migrate --reset flag 
```

### 5. Running FrontEnd
```shell
    cd FrontEnd/organicchain
    npm install
    npm run start
```
### 6. Connect your accounts to localhost. 

### Functional demo

### 1. Now Sign up on the webpage, by providing your name and Role. The selected account will be registered on Organic Chain.

### 2. Please keep an eye on Metamask to confirm the transaction to go through.

### 3. Now, you can sign in to the system and you will see UI based on your role. 

### 4. As, a FARMER you can view details of you all your farms on Home page and actions under Farmer page.

### 5. On farmer page you can add a farm to Organic Chain and tokenize a percentage of your farms produce.

### 6. Once you have added the farm by providing details Organic chain will hold the % of tokens for sale and you will recieve the rest of the token in you account.

### 7. As, an INVESTOR you will see your investments on Home page and options to invest in on Inverstor page.


As, a CONSUMER you can make use of the marketplace and buy authentic organic produce.
